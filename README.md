# lazy-media-server
Automated Plex/Jellyfin + *arr media stack
Selection based installs using docker

Work in progress, not production-ready


## Known Limitations:
- Minimal error handling
- Sensitive to typos and unexpected input
- May fail without clear error messages
- Not tested against edge cases
- Assumes specific environment/setup

## Usage notes
- Use at your own risk
- Do not rely on this script for critical tasks
- Review code before runnings if possible

## Contributing
Improvements to error handling, input validation, and robustness are very welcome.
